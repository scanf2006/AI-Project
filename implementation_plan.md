# 霓虹躲避者 (Neon Dodger) 实施计划

## 🛡️ 设计思想
- **视觉风格**: 采用深色背景搭配高饱和度的霓虹色彩 (蓝、粉、紫)。利用 CSS `backdrop-filter` 实现玻璃拟态效果。
- **用户体验**: 强调“灵敏度”与“反馈感”。每一次移动和碰撞都应有对应的视觉反馈。
- **技术选型**: HTML5 Canvas + 面向对象的 Vanilla JS。

## 📍 实施步骤

### 第一阶段：基础框架 (Foundation)
1. **HTML**: 定义画布容器和 UI 覆盖层 (Overlay)。
2. **CSS**: 
    - 利用 CSS 变量定义调色板。
    - 设置全屏背景渐变。
    - 定义 UI 动画 (入场、呼吸灯效果)。

### 第二阶段：游戏引擎核心 (Core Engine)
1. **Game 类**: 管理状态 (菜单、运行中、结束)。
2. **Entity 类**: 基类，处理位置、速度和绘制初步方法。
3. **Player 类**: 继承自 Entity，绑定输入事件。
4. **Enemy 类**: 继承自 Entity，处理随机生成与边界检测。

### 第三阶段：玩法与检测 (Gameplay & Physics)
1. 实现 `Spawner` 控制难度随时间增加。
2. 编写矩形或圆形碰撞检测函数。
3. 实现得分逻辑 (生存时间 = 分数)。

### 第四阶段：特效润色 (Polish)
1. **Particle 类**: 创建短暂存在的粒子，用于爆炸。
2. **Canvas 混合模式**: 使用 `globalCompositeOperation = 'lighter'` 增强发光感。
3. **UI 细节**: 添加平滑的数字滚动动画。

### 第五阶段：测试与优化 (Finalization)
1. 性能测试 (确保 60FPS)。
2. 移动端适配。
