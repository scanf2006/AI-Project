# 《星际竞技场》开发计划 - 淘汰制射击游戏

## 项目概述
一款基于 HTML5 Canvas 的俯视角射击游戏。玩家需要在不断缩小的安全区内击败所有敌人，成为最后的幸存者。

## 技术栈
- **核心**: HTML5 Canvas
- **逻辑**: 原生 JavaScript (ES6+)
- **样式**: 高级 CSS3 (包含霓虹特效、玻璃拟态)

## 核心功能规划
1.  **游戏引擎基础**
    - [ ] 游戏主循环 (requestAnimationFrame)
    - [ ] 输入处理 (键盘: WASD, 鼠标: 瞄准与射击)
    - [ ] 实体管理系统 (玩家、敌人、子弹、粒子)
2.  **视觉设计**
    - [ ] 动态背景设计（星空/网格）
    - [ ] 霓虹发光粒子系统（爆炸效果）
    - [ ] 平滑的相机跟随效果
3.  **核心玩法机制**
    - [ ] **淘汰机制**: 多个 AI 敌人，实时计数存活人数
    - [ ] **物理系统**: 碰撞检测 (圆形碰撞)
    - [ ] **缩圈机制**: 危险区域随时间从外向内收缩，停在其中会扣血
4.  **UI/UX 增强**
    - [ ] 极简风格的抬头显示 (HUD): 血条、存活人数、分数
    - [ ] 开始界面与死亡/获胜总结界面

## 开发流程
- **阶段 1**: 搭建基础 HTML/CSS 视觉框架 (预计 1 步)
- **阶段 2**: 实现核心移动与射击引擎 (预计 1 步)
- **阶段 3**: 实现 AI 逻辑与淘汰赛制 (预计 1 步)
- **阶段 4**: 加入缩圈与视觉特效抛光 (预计 1 步)
