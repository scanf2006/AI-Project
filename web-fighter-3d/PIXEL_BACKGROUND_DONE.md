## 背景特色

### 🎨 视觉元素

**远景层：**
- 深蓝色夜空渐变
- 建筑剪影（带随机窗户灯光）

**中景层：**
- 紫色建筑物
- 霓虹灯招牌（红色和青色）

**前景层：**
- 5个路灯（带光晕效果）
- 霓虹文字："FIGHT"、"格斗"、"KOF"
- 人群剪影（15个观众）
- 黄色地面透视线

### 🎮 像素化效果

背景使用了真正的像素化处理：
1. 先以高分辨率绘制（1024x512）
2. 缩小到像素分辨率（256x128）
3. 放大回原尺寸（保持像素块状）

这创造了经典的90年代街机游戏视觉效果！

## 技术细节

```javascript
// 纹理过滤设置（保持像素风格）
texture.minFilter = THREE.NearestFilter;
texture.magFilter = THREE.NearestFilter;

// 背景位置
position: (0, 20, -45)  // 在场景后方
size: 120 x 60 单位
```

## 如果想调整

### 调整背景位置
在 `createPixelArtBackground()` 函数中修改：
```javascript
background.position.set(0, 20, -45);  // x, y, z
```

### 调整背景大小
```javascript
const bgGeo = new THREE.PlaneGeometry(120, 60);  // 宽, 高
```

### 更改像素化程度
```javascript
pixelCanvas.width = 256;   // 降低 = 更像素化
pixelCanvas.height = 128;  // 降低 = 更像素化
```
