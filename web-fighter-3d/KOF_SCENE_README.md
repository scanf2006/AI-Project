# 拳皇风格场景说明

## 场景特色

这是一个完全重新设计的场景，灵感来自《拳皇》(The King of Fighters) 系列的经典街道场景。

## 场景元素

### 1. 夜晚氛围 🌙
- **深蓝色夜空** (0x1a1a2e)
- **夜晚雾效** - 营造神秘氛围
- **月光照明** - 蓝紫色的主光源

### 2. 霓虹灯招牌 ⚡ (KOF 标志性元素)
- **红色招牌**: "格斗" (左侧)
- **绿色招牌**: "FIGHT" (中央，最亮)
- **紫色招牌**: "拳皇" (右侧)
- **闪烁效果**: 随机闪烁模拟真实霓虹灯
- **彩色光源**: 每个招牌都有对应颜色的点光源

### 3. 观众剪影 👥 (经典KOF元素)
- **左右两侧各8人**
- **黑色剪影** - 只显示轮廓
- **随机高度** - 模拟不同姿势
- **随机位置** - 更自然的分布

### 4. 街道护栏 🚧
- **金黄色金属护栏**
- **左右两侧** - 定义格斗区域边界
- **高反光材质** - 金属质感

### 5. 街灯照明 💡
- **4盏街灯** - 前后左右对称分布
- **暖黄色光** - 营造温暖的街道氛围
- **球形灯头** - 现代路灯设计
- **长灯柱** (15米高)

### 6. 地面标线 📏
- **黄色中线** - 场地中央
- **前后边界线** - 标记格斗区域
- **半透明** - 不过分抢眼

### 7. 商店橱窗 🏪
- **6个发光橱窗**
- **橙黄色光** - 模拟店内灯光
- **半透明玻璃效果**

### 8. 建筑背景 🏢
- **25米高主建筑**
- **真实砖墙纹理**
- **70米宽** - 填满背景

### 9. 石板地面 🛣️
- **深灰色石板**
- **真实纹理贴图**
- **粗糙表面** - 街道质感

## 光照系统

### 主光源
- **月光** (DirectionalLight) - 蓝紫色，模拟夜晚月光

### 点光源
- **霓虹灯** (3个) - 红/绿/紫
- **街灯** (4个) - 暖黄色
- **环境光** - 深蓝色基础照明

## 与拳皇的相似之处

✅ **夜晚街道** - KOF经典场景时间
✅ **霓虹灯招牌** - 日本/中国街道特色
✅ **观众围观** - 格斗游戏标配
✅ **城市氛围** - 繁华都市背景
✅ **彩色光效** - 霓虹灯的多彩光照

## 技术亮点

- **动态闪烁**: 霓虹灯使用 setInterval 实现随机闪烁
- **随机生成**: 观众位置和高度随机，每次刷新都不同
- **多层光照**: 结合环境光、方向光和多个点光源
- **真实纹理**: 地面和墙壁使用在线照片纹理
- **阴影系统**: 所有主要物体都启用阴影

## 性能优化

- 观众使用简单的 Box 几何体（低多边形）
- 纹理异步加载，不阻塞游戏启动
- 合理的光源数量（避免过多点光源）
