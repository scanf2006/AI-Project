# 3D 模型文件说明

## 已下载的模型

本文件夹包含游戏所需的 3D 模型文件：

### 1. RobotExpressive.glb (453 KB)
- **用途**: Player 1 (玩家角色)
- **动画**: Idle (待机), Running (跑动), Punch (出拳)
- **缩放**: 0.8x
- **特点**: 可爱的黄色机器人，动作丰富

### 2. Soldier.glb (2.1 MB)
- **用途**: Player 2 (AI 对手)
- **动画**: Idle (待机), Run (跑动), Walk (行走)
- **缩放**: 2.5x
- **特点**: 特种兵模型，使用"肩撞"作为攻击动作

## 优势

✅ **无需网络加载** - 游戏启动更快
✅ **离线可用** - 没有网络也能玩
✅ **稳定性更好** - 不受网络波动影响

## 来源

这些模型来自 Three.js 官方示例库，遵循开源许可。

## 重新下载

如果需要重新下载模型文件，运行：
